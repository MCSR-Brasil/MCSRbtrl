Static webpage for a OBS Browser Source to automatically play livestreams based on who has the best pace using a twitch embed, used for 24/7 BTRLS

- Static page, you can dowload the html and use it if you want
- Changes runer based on best pace, configured with some weights on JS
- Changes runner name based on twitch name and PB based on a spreadsheet, so we can set pbs that arent on Paceman
- Easy to add and change overlays
- Just read the README.md

They are supposed to go into a OBS Browser Source 1920x1080, otherwise they might look goofy, i havent tested
